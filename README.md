# Personal Portfolio Website - Duy Tran
A responsive, modern personal portfolio built with HTML/CSS/JavaScript to showcase projects, skills, and experience in a clean, friendly format.
**Website:** [notduey.github.io](https://notduey.github.io)

## Overview
This project serves as a front-end portfolio and technical showcase.
It focuses on modern design, responsive navigation, UI state management, DOM manipulation, accessibility-minded interactions, client-side validation, and async form submission.

## Key Features
- Responsive navigation with mobile menu toggles
- Scroll-based active section highlighting
- Dark / light theme toggle with persistent user preference
- Skills accordion UI
- Qualifications tab system (Education / Experience)
- Portfolio and expertise sliders
- Modal windows for content
- Copy-to-clipboard email interaction
- Contact form with validation and async submission
- Consistent layout, spacing, and typography using CSS variables

## Installing/Locally Hosting the Portfolio
**1. Clone the repository:**  

*SSH*
```bash
git clone git@github.com:notduey/notduey.github.io.git
```
*HTTPS*
```bash
https://github.com/notduey/notduey.github.io.git
```
**2. CD into the directory:**
```bash
cd.../.../notduey.github.io
```
**3. Open the index file on your browser:**  

*Windows*
```bash
start index.html
```
*MacOs*
```bash
open index.html
```
